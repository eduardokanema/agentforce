# Project Harness V1

Project Harness V1 is the new primary operator surface for AgentForce.

## Goal

Unify planning, launch, execution, evidence, and history around one repo-root project view.

## Hero Story

An engineering lead opens a project and can answer three questions immediately:

- What is happening now?
- What is next?
- What proof do we have?

## Non-Goals

- persisted data migration
- Black Hole-first rollout
- replacing existing mission execution runtime

## V1 Scope

- projects inbox
- project cockpit
- cycle lineage
- compact evidence summaries
- docs and status as explicit system-of-record
